== Brasileirao2015
Web application with teams and players of the Brazilian soccer championship.

== Installation

$ git clone https://github.com/jessicalins/brasileirao2015.git

$ gem uninstall libv8 -a -x

$ bundle install

$ gem install therubyracer -v '0.12.0'

== Database creation

$ rake db:create

$ rake db:migrate
